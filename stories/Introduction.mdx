# Shadcn Datetime Picker

**Timezone-aware, keyboard-first date and time pickers for [shadcn/ui](https://ui.shadcn.com).**

Three single-file components you copy into your project and modify directly. No runtime dependency on this package — `npx shadcn add` drops the source into your `components/` folder and you take ownership from there.

## Install

Pick the component you need and run one command. The `shadcn` CLI drops the source file into your `components/` folder and installs the shadcn primitives and npm peers each component needs — see each component's docs page for the exact list.

### DateTimePicker — full date + time popover

<Source language="bash" code={`npx shadcn@latest add https://shadcn-datetime-picker-pro.vercel.app/r/datetime-picker.json`} />

### DateTimeInput — segmented keyboard input

<Source language="bash" code={`npx shadcn@latest add https://shadcn-datetime-picker-pro.vercel.app/r/datetime-input.json`} />

### SimpleTimePicker — time-only dropdown

<Source language="bash" code={`npx shadcn@latest add https://shadcn-datetime-picker-pro.vercel.app/r/simple-time-picker.json`} />

> Prefer the manual route? Each component is one file — copy from the [GitHub repo](https://github.com/huybuidac/shadcn-datetime-picker/tree/main/components) into your `components/` folder.

## Highlights

- **Timezone-aware** — `DateTimePicker` and `DateTimeInput` render and edit any IANA timezone via `react-day-picker`'s `TZDate` (`SimpleTimePicker` is local-time only).
- **Keyboard-first** — `DateTimeInput`'s segments step with arrow keys, digit keys auto-advance, type `a`/`p` to toggle AM/PM.
- **Soft-keyboard ready** — handles Android Gboard, IME composition, and `beforeinput` quirks.
- **Min/Max constraints** — `DateTimePicker` greys out out-of-range dates and times; `SimpleTimePicker` disables out-of-range hour/minute/second options. (`DateTimeInput` itself doesn't enforce a range — pair it with a picker if you need bounds.)
- **Custom trigger** — render `DateTimePicker` behind any button, badge, or your own component via `renderTrigger`.
- **Form-friendly** — all three work with `react-hook-form`; `DateTimeInput` additionally forwards refs and renders an inline error message for native form integration.
- **Fully styled with Tailwind** — uses your project's `components.json` aliases and CSS variables.
- **Copy-paste ownership** — three single-file components, no runtime dependency on this package.
